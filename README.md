# Blinkit Sales Analytics Dashboard | Power BI

This **Blinkit Sales Analytics Dashboard** is an interactive Power BI solution built to analyze grocery retail performance across multiple outlets, product categories, and locations. It provides a complete view of sales, item movement, outlet efficiency, and customer satisfaction, helping businesses make faster, smarter retail decisions.

---

## What this dashboard delivers

This dashboard offers a **data-driven overview of Blinkitâ€™s retail operations** by combining sales, item volume, ratings, and outlet performance into a single analytical platform.

It enables users to:

- Track total sales and average sales performance  
- Monitor number of items sold across categories  
- Analyze outlet performance by size, type, and location  
- Evaluate customer satisfaction using ratings  
- Identify high-performing and underperforming product segments  

---

## Key Performance Indicators (KPIs)

The dashboard highlights critical retail metrics such as:

- **Total Sales**  
- **Average Sales per Item**  
- **Number of Items Sold**  
- **Average Customer Rating**

These KPIs give an instant snapshot of overall business performance.

---

## Key Visual Insights

### Category & Product Performance
- **Sales by Category** identifies which grocery segments (fruits, snacks, dairy, household, etc.) generate the most revenue.

### Outlet Performance
- **Sales by Outlet Size** compares performance of small, medium, and large stores.  
- **Sales by Outlet Location Tier** reveals how Tier 1, Tier 2, and Tier 3 cities perform.  
- **Sales by Outlet Type** compares supermarkets and grocery stores across sales, volume, and ratings.

### Trend Analysis
- **Outlet Establishment Trend** shows how sales performance has changed over time based on store opening years.
